Head to ICU. 是 Francie Jiang 使用她刚学的 Vue.js 和 Tailwind CSS 写出来的垃圾东西（

论具体来由的话，就是她觉得自己这个精神状态（或者健康状况）迟早要被送去 ICU，于是她就去搜了 headto.icu 这个域名，然后发现它一年只要十块钱，于是直接去注册了，然后把这个玩意部署在了 Vercel 上面。

但是因为 Francie 太菜惹，所以她其实又写了一个 Python 的程序用来生成存储 post 内容的 postContent.js 。
